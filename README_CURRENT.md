## 🚀 快速开始

### 1分钟测试

```bash
# 1. 克隆
git clone https://github.com/zhaoyuong/clawdbot-python.git
cd clawdbot-python

# 2. 安装
poetry install

# 3. 配置
cp .env.example .env
nano .env  # 添加API密钥

# 4. 测试
poetry run clawdbot agent --message "Hello!"
```

### Docker测试

```bash
# 安全的Docker测试
chmod +x test-docker-safe.sh
./test-docker-safe.sh
```
